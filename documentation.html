ScalaCheck Documentation

User Guide

The ScalaCheck User Guide is developed continuously, and should reflect the latest release of ScalaCheck reasonably well. If you find any inconsistencies or missing parts, please report it (or submit a pull request!).

API Documentation

The API documentation for the latest ScalaCheck release is available both online and packaged for download. You can find documentation for other ScalaCheck releases here.

ScalaCheck: The Definitive Guide

ScalaCheck: The Definitive Guide is written by Rickard Nilsson, the creator of ScalaCheck. The book is a complete guide to ScalaCheck, as well as a thorough introduction to the concept of property-based testing. There is a resource page available for the book, where you can browse code examples and view/submit errata.
